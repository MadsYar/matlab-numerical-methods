# MATLAB Codes Collection

This repository contains various MATLAB codes developed over the years for numerical methods, computational mathematics and scientific computing purposes. These implementations were created for coursework, research and exploration of numerical algorithms.

## File Descriptions

### Numerical Root Finding
- **Newton.m** - Implementation of Newton's method for solving scalar equations f(x)=0 with a starting guess and specified number of iterations.
- **Newtonsys.m** - Newton's method for solving systems of nonlinear equations F(x)=0 using Jacobian matrices.
- **TestNewton.m** - Test script demonstrating Newton's method on example polynomial equations.
- **Kvadkonv.m** - Analysis script for testing and verifying quadratic convergence properties of Newton's method.
- **funFdF.m** - Example function defining a system of nonlinear equations and its Jacobian for use with Newtonsys.

### Interpolation
- **CardinalPolynomial.m** - Computes values of Lagrange cardinal polynomials for a given set of nodes.
- **InterpolerLagrangeForm.m** - Calculates interpolating polynomial values using Lagrange form representation.

### Numerical Integration
- **MySimpson.m** - Implementation of the composite Simpson's rule for numerical integration of functions over an interval.

### Ordinary Differential Equations
- **MyRK4System.m** - Classical fourth-order Runge-Kutta method (RK4) for solving systems of ODEs with specified step size.
- **odefun.m** - ODE system definition for what appears to be orbital mechanics or gravitational problems.
- **odefun2.m** - Another ODE system definition for a second-order differential equation.
- **phi.m** - Implementation of a shooting method for solving boundary value problems using ODE solvers.

### Utility
- **sd.m** - Simple utility script for clearing workspace variables and command window.
